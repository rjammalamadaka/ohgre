These terms and conditions ("terms") apply to the Georgia Natural Gas ("GNG") Refer-a-Friend program ("RAF Program" or "Program"). Referrers must click "Agree" to the RAF Program terms to participate in the RAF Program. Referees agree to the RAF Program terms by using an RAF Program Referral Code when enrolling with GNG.
